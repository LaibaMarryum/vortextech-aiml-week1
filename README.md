# VortexTech AI & ML Internship - Week 1

## Titanic Dataset: Data Cleaning and Exploratory Data Analysis (EDA)

### Project Overview
This project was completed as part of the VortexTech AI & ML Internship Week 1 assignment. The objective was to perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset using Python.

### Dataset
- Titanic Dataset (`train.csv`)

### Objectives
- Load the dataset using Pandas.
- Explore the dataset structure.
- Identify missing values.
- Clean the data by handling missing values.
- Remove unnecessary columns.
- Perform exploratory data analysis.
- Visualize relationships between different features.

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

### Data Cleaning
- Removed unnecessary columns.
- Filled missing values in the `Age` column using the median.
- Filled missing values in the `Embarked` column using the mode.
- Checked for duplicate and missing values.

### Exploratory Data Analysis
- Examined passenger demographics.
- Analyzed survival rates.
- Explored relationships between passenger class, gender, age, and survival.
- Generated visualizations to better understand the dataset.

### Files Included
- `Week1.ipynb`
- `README.md`

### Outcome
The dataset was successfully cleaned and analyzed, providing insights into the factors that influenced passenger survival on the Titanic.
